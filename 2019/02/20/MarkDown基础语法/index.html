本文介绍了一种文本标记语言MarkDown的基础语法

#####Markdown是什么？
它是一种纯文本格式标记语言。

#####目的
通过markdown可以使普通文本具有一定的格式，便于阅读。

#####输入的是什么？
1.普通文本
2.语法符号

#####优点：
1.纯文本，只要支持markdown就可以获得一样的格式效果
2.操纵简单。

#####缺点：
1.需要记忆一些语法
2.有些平台不支持Markdown编辑模式

#一、标题
语法：#*n(1~6）+ title
# - 语法符号
title - 标题内容
注：最长支持6级标题

#二、字体

  • 斜体
    语法：*要倾斜内容* - 内容前后加1个*号
  • 加粗
    语法：**要加粗内容** - 内容前后加2个*号
  • 斜体加粗
    语法：***要斜体加粗内容*** - 前后加3个*号
  • 删除线
    语法：~~删除线内容~~ - 前后加2个~号
