#一、生成式
1.什么是生成式
生成式就是生成器的一种特殊写法
作用：让变量去序列中取值，每取一个值，就将对应的表达式的值，作为生成器的元素
2.写法
a.生成器 = （表达式 for 变量 in 序列）

#二、文件操作
1.数据本地化和数据持久化
定义：将数据以文件的形式存到计算机硬盘（程序中保存的数据，当程序运行结束会自动销毁）

2.如何数据持久化（如何将程序中数据和本地文件数据关联）
文件读写操作：
读：获取文件中的内容
写：将数据添加到文件中

a.文件操作的基本流程：
第一步：打开文件
第二步：文件操作（读/写）
第三步：关闭文件

3.打开文件
open(file, mode=’r’, encoding=None)
功能：以指定的模式打开指定的文件，并且返回被打开的文件对象
说明：
file:字符串，需要打开的文件的路径
1.（了解）绝对路径：文件在电脑上的全路径
2.（掌握）相对路径：相对当前代码文件对应的目录来确定的路径
./   当前py文件所在的目录
../   代表当前目录的上层目录
…/  代表当前目录的上层目录的上层目录
mode:字符串，文件的打开方式
r: 以只读的形式打开，文本 （默认值）
w: 以写的形式打开（覆盖），文本
a:以写的形式打开（追加），文本
rb/br : 以只读的形式打开，二进制
wb/bw: 以写的形式打开（覆盖），二进制
+:以读写的形式打开
注意：
1.以读的方式打开一个不存在的文件，程序报错：FileNotFoundError
2.以写的方式打开一个不存在的文件，不会报错，还会自动创建这个文件
encoding:字符串，文字编码方式
      utf-8:支持所有的语言
     gbk:只支持英文
注意：
a.读和写对应的编码要保持一致
b.非文本文件不能设置编码encoding，例如：图片，视频，可执行文件等..
c.传值时，必须使用关键字参数
d.以二进制形式打开文件，不能设置encoding

4.操作文件：
a.读操作
文件对象.read()
目的：获取指定文件的内容
注意：打开一次，只能读一次，再用需要再打开

5.关闭文件
文件对象.close()

6.文件操作的简写方式
说明：打开文件后，操作完成后会自动关闭文件
